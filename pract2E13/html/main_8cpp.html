Detailed Description

Main program for vehicle management.

This program allows the user to enter vehicle data, store it in a dynamic array, show them on the screen, search for a vehicle by license plate, find the most expensive one and load vehicles on a transport. Exceptions are handled in case of errors in data entry.

Returns 0 if the execution is successful.
